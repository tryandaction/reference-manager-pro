## 🚀 如何使用

### 步骤 1：先读商业化指南（30 分钟）

打开 `BUSINESS_GUIDE.md`，重点阅读：
- 第 1 章：商业化基础概念（了解基本术语）
- 第 2 章：收入模式详解（理解免费增值）
- 第 4 章：支付平台选择（学习如何用 Gumroad）
- 第 6 章：营销推广（知道在哪里推广）

**目标**：对商业化有基本概念，不再迷茫

### 步骤 2：开始执行开发任务（每天 1-2 小时）

打开 `DEVELOPMENT_ROADMAP.md`，从阶段 1 开始：

#### 任务执行流程
1. **选择一个任务**（按顺序，从任务 1.1 开始）
2. **复制 [AI 提示] 部分**
3. **粘贴给 AI**（Claude/ChatGPT/Windsurf）
4. **AI 会生成代码/内容**
5. **按照 [验证清单] 检查结果**
6. **在任务前打勾 ✅**
7. **继续下一个任务**

#### 示例：执行任务 1.1（创建图标）

```
1. 打开 DEVELOPMENT_ROADMAP.md
2. 找到"任务 1.1：创建插件图标"
3. 复制这段文字给 AI：

我需要为 VS Code 插件创建一个图标。插件名称是 "Reference Manager Pro"，是一个 LaTeX 文献管理工具。

要求：
1. 使用 SVG 格式
2. 尺寸：128x128px
3. 设计元素：书籍/引用符号/AI 元素
4. 颜色：专业的学术风格（蓝色或深色系）
5. 简洁现代

请生成 SVG 代码，我可以直接保存为 icon.svg 文件。
同时生成 PNG 版本的转换命令。

4. AI 会生成 SVG 代码
5. 保存为 images/icon.svg
6. 验证图标是否清晰
7. 打勾 ✅
```

### 步骤 3：遇到问题时查阅商业化指南

开发过程中有疑问？回到 `BUSINESS_GUIDE.md` 查找答案：

- **不知道如何定价？** → 看第 3 章
- **不知道如何设置 Gumroad？** → 看第 4.2 节
- **不知道如何营销？** → 看第 6 章
- **不知道如何处理退款？** → 看第 8.2 节

## 💡 重要提示

### ✅ 要做的事
1. **按顺序执行任务**（不要跳过）
2. **每完成一个任务就打勾**（追踪进度）
3. **遇到问题先查文档**（90% 的问题文档里都有答案）
4. **保持耐心**（商业化需要时间）

### ❌ 不要做的事
1. **不要追求完美**（完成比完美重要）
2. **不要一次做太多**（每天 1-2 个任务即可）
3. **不要害怕失败**（这是学习过程）
4. **不要放弃**（坚持 3 个月才能看到效果）
